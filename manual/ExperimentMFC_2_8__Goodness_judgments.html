ExperimentMFC 2.8. Goodness judgments

If numberOfGoodnessCategories is not 0, some more rectangles will be drawn, as in the example experiment. You specify again the locations of these rectangles (in the example, they touch each other), and the texts on them. Praat will record the number of the button when the listener clicks on it. Thus, if she clicks on the button labelled "1 (poor)", Praat will record a goodness judgment of 1, because this is the first button in the list. If she clicks on "5 (good)", Praat will record a goodness judgment of 5.
