Add action command...

One of the hidden commands in the Praat menu of the Object window. With this command, you add a button to the dynamic menu in the Object window.

Settings

See Add to dynamic menu....

Usage

You can use this command in your initialization script or in plug-ins.

Normally, however, if you want to add a command to the dynamic menu, you would use the command Add to dynamic menu... of the ScriptEditor instead.
