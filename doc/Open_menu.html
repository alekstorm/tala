Open menu

One of the menus in the Object window.

With the Open menu, you read one or more objects from a file on disk into memory. The resulting object(s) will appear in the List of Objects.

The Open menu contains the command Read from file..., which recognizes most file types, and perhaps several other commands for reading unrecognizable file types (e.g., raw sound data), or for interpreting known file types in a different way (e.g., reading two mono sounds from one stereo sound file):
