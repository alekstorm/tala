ChebyshevSeries

One of the types of objects in PRAAT.

An object of type ChebyshevSeries represents a linear combination of Chebyshev polynomials Tk(x).

ChebyshevSeries (x) = Σk=1..numberOfCoefficients ck Tk(x)
