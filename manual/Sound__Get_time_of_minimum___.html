Sound: Get time of minimum...

A command available in the Query menu if you select a Sound object. The Info window will show the time (in seconds) associated with the minimum pressure in a specified time range.

Settings

Time range (s)
the time range (t1, t2). Values outside this range are ignored, except for purposes of interpolation. If t1 is not less than t2, the entire time domain of the sound is considered.
Interpolation
the interpolation method (None, Parabolic, Cubic, Sinc) of the vector peak interpolation. The standard is Sinc70 because a Sound object is normally a sampled band-limited signal, which can be seen as a sum of sinc functions.
