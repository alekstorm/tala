Sound: To Polygon...

A command that creates a Polygon from each selected Sound, where the Polygon's (x,y) points are defined by the (time, amplitude) pairs of the sound.

Settings

Channel
defines which channel of the sound is used.
Time range (s)
defines the part of the sound whose (time, amplitude) pairs have to be included.
Vertical range
defines the vertical limits, larger amplitudes will be clipped.
Connect at
defines the amplitude where the start and the end point have to be connected.

Details

The (x,y) points in the Polygon are calculated as follows:

The first point is at (tmin,connectAt), the second point at (tmin,sound(tmin)), then the selected sample points of the sound are added as (time,value) pairs. Next the point (tmax,sound(tmax)) and (tmax,connectAt) are added. Finally the Polygon is closed by adding the point (tmin, closeAt). Here tmin and tmax are minimum and maximum value of the selected part of the sound, sound(tmin) and sound(tmax) are the interpolated amplitudes of the sound at tmin and tmax, respectively, and, closeAt is the user defined amplitude value of the connection point.

Example

The following script paints the area under a sound curve in red and the area above in green.

s = Create Sound from formula... s Mono 0 1 10000 0.5*sin(2*pi*5*x)
 #Connection point is at amplitude -1: area under the curve.
 p1 = To Polygon... Average 0 0 -1
 Paint... {1,0,0} 0 0 -1 1
 select s
 #Connection point is at amplitude 1: area above the curve.
 p2 = To Polygon... Average 0 0 1
 Paint... {0,1,0} 0 0 -1 1

s = Create Sound from formula... s Mono 0 1 10000 0.5*sin(2*pi*5*x) p1 = To Polygon... Average 0 0 -1 Paint... {1,0,0} 0 0 -1 1 Remove select s p2 = To Polygon... Average 0 0 1 Paint... {0,1,0} 0 0 -1 1 plus s Remove
