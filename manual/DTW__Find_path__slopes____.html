DTW: Find path (slopes)...

Finds the optimal path for the selected DTW satisfying slope constraints.

Number of non-diagonal steps, Number of diagonal steps
implement a slope constraint. Every Number of non-diagonal steps steps in a X or Y direction must be followed by at least Number of diagonal steps steps in the diagonal direction. By setting Number of non-diagonal steps to 0 you explicitly forbid non-diagonal steps, while setting Number of diagonal steps to 0 does not impose any slope constraints at all.
X weight, Y weight, Diagonal weight
determine the weights in the cumulative distance calculation.

Examples

For the constraint 1/2 <= slope <= 2, set nonDiagonalSteps = 2 and diagonalSteps = 1.

For the constraint 1/3 <= slope <= 3, set nonDiagonalSteps = 3 and diagonalSteps = 1.

To visualize the slope constraints, select a DTW and chose To Polygon (slopes).... Next select Draw... (don't forget to make the drawing domains equal to the domains of the two objects in the DTW).

For more information see the article of Sakoe & Chiba (1978).
