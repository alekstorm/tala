Sound files 4. Files that Praat can write

Praat can write five types of standard sound files in an appropriate linear 16-bit formats, and a number of proprietary types of sound files as well:

  •  Save as WAV file... (16-bit little-endian)
  •  Save as AIFF file... (16-bit big-endian)
  •  Save as AIFC file... (16-bit big-endian)
  •  Save as NeXT/Sun file... (16-bit big-endian)
  •  Save as NIST file... (16-bit little-endian)
  •  Save as FLAC file... (16-bit)
