Fixed menu commands

The commands in the fixed menus of the Object window (Praat, New, Open, Help, Goodies, and Preferences) and the Picture window (File, Edit, Margins, World, Select, Pen, Font, Help).

These commands are always clickable (if not hidden) and scriptable (if not added).
