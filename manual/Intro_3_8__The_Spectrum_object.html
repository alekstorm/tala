Intro 3.8. The Spectrum object

To compute a Fourier frequency spectrum of an entire sound, select a Sound object and choose To Spectrum... from the Spectrum menu. A new Spectrum object will appear in the List of Objects. To view or modify it (or listen to its parts), click View & Edit. To print it, choose one of the Draw commands to draw the Spectrum object to the Picture window first.
