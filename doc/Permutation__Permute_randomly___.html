Permutation: Permute randomly...

Generates a random permutation for the elements in the given range.

Setting

Index range
defines the range of elements that will be permuted.

Example

With Index range = [3,6], the permutation (1,2,3,4,5,6,7) is turned into one of 24 (= 4!) possible permutations, for example (1,2,(4,5,3,6),7).
