Open Praat script...

A command in the Praat menu for editing an existing Praat script. It creates a ScriptEditor and asks you to select a file. If you click OK, the file is read into the ScriptEditor window, and you can run and edit it; if you click Cancel, you get an empty script, as with New Praat script.
