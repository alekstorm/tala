Procrustes transform

A transformation that only uses a combination of a translation, a scaling and a rigid transformation to transform one Configuration such that it matches as closely as possible another Configuration.

We speak of orthogonal Procrustes transform when only the rigid transformation is allowed but no scaling or translation.

For more information about the Procrustes transform and its algorithm see chapter 19 in Borg & Groenen (1997).
