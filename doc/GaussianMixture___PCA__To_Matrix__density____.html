GaussianMixture & PCA: To Matrix (density)...

Represent the GaussianMixture p.d.f. on the plane spanned by PCA. This makes it possible to draw the p.d.f. as grey values with one of the special Matrix image drawing methods.

For each cell in the matrix of dimension numberOfRows × numberOfColumns, the p.d.f. will be evaluated.
