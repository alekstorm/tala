Ltas: Get frequency from bin number...

A query to the selected Ltas object.

Return value

the frequency (in Hz) associated with a specified bin number.

Setting

Bin number
the bin number whose frequency is sought.

Algorithm

the result is

f1 + (binNumber - 1) · Δf

where f1 is the frequency associated with the centre of the first bin, and Δf is the bin width.
